Step 3: Configure MongoDB for External Access

Open the MongoDB configuration file for editing:

sudo nano /etc/mongod.conf

Change the bindIP in the configuration file to 0.0.0.0 to accept data from your IP address, and ensure it listens on port 8080 as well. Save the changes and exit the editor.
